- Для 32-х разрядных версий ОС Windows, использовать исходные файлы папки Win32
- Для 64-х разрядных версий ОС Windows, использовать исходные файлы папки x64
Для 64-х разрядных версий ОС Windows необходимо отключить проверку цифровой подписи драйвера
Для этого, откройте консоль под привилегиями администратора и выполните команду
bcdedit -set TESTSIGNING ON
Выполнить перезагрузку

1. Перейти в папку с исходными файлами
2. Перейти в подпапку Release
3. Щелкнуть два раза по файлу ReadOnlyness.cer и выполните его установку с автоматическим определением места хранения.
Данный файл представляет корневой самоподписной сертификат, используемый для проверки подписи драйвера из каталога безопасности.
4. Перейти в подпаку ReadOnlyness
5. Вызвать контекстное меню на файле ReadOnlyness.inf и выбрать пункт "Установить".
6. Выполнить в консоли администратора загрузку драйвера:
fltmc load readonlyness
7. Перейти на две папки вверх
8. Создать в блокноте список правил фильтрации в виде и сохранить его под именем input.txt рядом с загрузчиком правил ReadOnlyRuleLoader.exe, например:
C:\Program Files\1.txt
C:\*\test.txt
*.myext
9. Выполнить из консоли ReadOnlyRuleLoader.exe. Должно быть выдано сообщение об успешной загрузке.


Поддерживаемые ОС - WinXP-Win10 x86/x64
